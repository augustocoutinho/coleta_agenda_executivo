# coleta_agenda_executivo

Algoritmo que facilita a coleta de agenda do Presidente Jair Bolsonaro. O script conta com dois parâmetros, ambos que fazem menção à data: (1) o primeiro dia de coleta; e (2) o último.

O código retorna uma base de dados contendo o nome da autoridade – no caso, o Presidente Jair Bolsonaro –, o local, data, horário e o título do compromisso.
